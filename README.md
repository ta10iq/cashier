# نظام كاشير عربي

تطبيق نقطة بيع بسيط يعمل ببايثون فقط بدون تثبيت مكتبات خارجية. يستخدم SQLite كقاعدة بيانات محلية مؤقتة، ويدعم:

- إضافة المنتجات وتعديلها وحذفها.
- البحث في المنتجات.
- بيع المنتجات من شاشة كاشير.
- خصم المخزون تلقائيًا بعد البيع.
- تسجيل آخر المبيعات مع الإجمالي والمدفوع والباقي.

## التشغيل

من داخل مجلد المشروع شغّل:

```powershell
python app.py
```

أو شغّل السكربت الجاهز كمسؤول على ويندوز:

```cmd
setup_and_run_cashier.cmd
```

لتهيئة IIS كرابط بدون بورت عبر دومين مجاني مقترح:

```cmd
setup_iis_reverse_proxy.cmd
```

افتح المتصفح على:

```text
http://127.0.0.1:3737
```

سيتم إنشاء ملف قاعدة البيانات تلقائيًا باسم:

```text
cashier.sqlite3
```

## الاختبار

لتجربة العمليات الأساسية:

```powershell
python -m unittest test_app.py
```
